Introduction

A stroke is a medical emergency where blood flow to the brain is dangerously restricted, and one in four people over the age of 25 will have one in their lifetime. If not treated properly, aftereffects may include restricted motion, paralysis, and difficulty communicating. This is why we decided it was important to raise awareness of if any individual may be at higher risk for a stroke. This project uses data analysis to calculate stroke probability based on a few key factors, such as BMI, gender, relationship status (ever married) and age.

Logistic Regression

Logistic regression was the first binary predictor we used. It models probability (whether something will or won't happen) based on processed input, and reads like a Punnett Square. For our data set, logistic regression had a relatively low accuracy of 54%. However, the recall rate was high (number of correctly predicted positives divided by total number of correctly predicted strokes or no-strokes).

Decison Tree Classifier

The Decision Tree Classifier was created around the 1960s, however a similar concept has been used since the earliest written records. The decision tree uses a tree like structure, and possible combinations of data to solve a particular problem. A decision tree is a kind of supervised machine learning that categorizes outcomes based on the answers to a previous set of questions. In terms of accuracy, the Decision Tree Classifier did almost as well as the Multi-Layer Perceptron, with just 8% lower accuracy (80%). However, the recall rate was relatively low.

Multi-Layer Perceptron

Multi-Layer Perceptron is another supervised learning model that uses layers, functions, and connections to predict an output. In order to function, we need to train data sets. If given data sets, it can function as a nonlinear function approximator for either classification or regression. This differs from logistic regression because there can be hidden layers. Conclusion

From exploratory data analysis, it was observed that the features that had the highest impact on having a stroke was age, average glucose level, and being married. The proportion of individuals that have a stroke increases greatly after the age of 40. Individuals with higher average glucose levels and individuals that are married both have an increased risk of having a stroke.
Regarding model predictions, it was difficult to improve the accuracy past the null accuracy for this dataset. This dataset was relatively small (5000 samples) with a highly imbalanced target distribution (about 5% minority class). Since models perform only as well as the quality of the data, the main improvement for future experiments in the topic of stroke predictions would be to increase the quantity and diversity of data. Specifically, more data would be helpful and more data with positive stroke labels to balance this dataset would be helpful.
The applicability of our models depend on the use case for stroke diagnoses. If one wants to use a model with extremely high accuracy, a null predictor (predicting everyone to not have a stroke) will be correct 96% of the time. However, if one wants to capture the population that will have a stroke while not caring about false positives, the logistic regression model performed best with a 0.98 recall in the has stroke category.
